## Overview

**LinuxQuest** is a gamified, story‑driven platform that teaches real‑world Linux administration through missions, competitions, and skill‑based progression — from `ls` to `kubectl`.

---

## Why LinuxQuest?

Traditional tutorials teach commands in isolation. LinuxQuest integrates each command into a narrative where you *need* the command to advance, ensuring lasting retention.

---

## Quick Start (Bootcamp)

1. Install the CLI (or use the hosted demo).
2. Run `linuxquest bootcamp` to begin the 10‑minute interactive tutorial.
3. Complete the basic tasks (terminal basics, navigation, file creation, pipes, etc.).

The bootcamp unlocks the rest of the platform.

---

## Core Features

- **Skill Trees** – Organized groups of related commands (Core Linux, Networking, Security, DevOps, Performance).
- **Campaigns** – Narrative story arcs (e.g., *Operation Antariksha*). Each chapter maps to a real‑world scenario.
- **Tracks** – Stand‑alone learning paths (Shell Scripting, Storage, Package Management, Vim, Git, Docker, Kubernetes, Environment Variables).
- **Daily Challenges** – Short, LeetCode‑style tasks that refresh daily.
- **Weekly Quests** – Time‑limited, competitive missions with leaderboards.
- **Rating System** – Global Linux Elo rating (Newcomer → Wizard) that reflects skill mastery.

## Tracks (Highlights)

- **Shell Scripting (NEW)** – Defensive scripts, cron integration, JSON handling.
- **Storage & Filesystems** – `df`, `du`, `mount`, `fsck`, partition tools.
- **Package Management** – `apt`, `dnf`, `rpm`, `dpkg`.
- **Vim** – Basics and mastery modes.
- **Git** – Recovery, rebasing, bisect.
- **Docker** – Container inspection, logs, compose.
- **Kubernetes (Advanced)** – Pod management, taints.
- **Environment Variables** – Secure secret handling.

---

## Difficulty Levels

- **Standard** – Straightforward command usage.
- **Hard** – Complex pipelines, regex, and performance optimizations.

---

## Daily Challenges & Weekly Quests

- **Daily** – Quick 5‑10 min tasks (e.g., “Find all 500 xx responses in a 500 MB nginx log”).
- **Weekly** – Multi‑hour themed quests with global leaderboards.

## Seasonal Events

Special limited‑time missions (e.g., *LinuxFest 2027*) with exclusive badges and community‑voted themes.

## Rating & Leaderboard

Earn XP, increase your Elo rating, and climb the global leaderboard. Your profile shows level, skill breakdown, and recent badges.
